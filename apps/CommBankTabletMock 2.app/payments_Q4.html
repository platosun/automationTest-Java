To make a payment to someone new using the CommBank app for tablet, go to the accounts 'dashboard' or account details page and:

  1. Select Pay button
  2. Select Pay Someone button
  3. Select "Add person +"
  4. Enter account name
  5. Enter BSB Number
  6. Enter Account Number
  7. Select Next
  8. Enter amount
  9. Enter description
  10. Select Pay

To make a payment to someone in my address book using the CommBank app for tablet, go to the accounts 'dashboard' or account details page and:

  1. Select Pay button
  2. Select Pay Someone button
  3. Select the contact you wish to pay
  4. Enter amount
  5. Enter description
  6. Select Pay

To make a payment to new BPAY biller using the CommBank app for tablet, go to the accounts 'dashboard' or account details page and:

  1. Select Pay button
  2. Select BPAY button
  3. Select "Add BPAY biller +"
  4. Enter Biller Code
  5. Enter Reference Number
  6. Enter Biller Name
  7. Select Next
  8. Enter amount
  9. Enter description
  10. Select Pay

To make a payment to an existing BPAY biller using the CommBank app for tablet, go to the accounts 'dashboard' or account details page and:

  1. Select Pay button
  2. Select BPAY button
  3. Select the BPAY biller contact you wish to pay
  4. Enter amount
  5. Enter description
  6. Select Pay
